## Connection scheme ALEX

Laser green:

* Digital:
  + APD (sampling): 6612 Counters 0 and 1, Source
  + AOTF: 6612 Counter 4 Out

* Analog: 6713 AO.1

Laser red:

* Digital:
  + APD (sampling): 6612 Counters 2 and 3, Source
  + AOTF: 6612 Counter 5 Out

* Analog: 6713 AO.0

APD gate signals:

* 6612 Counters 6 and 7, Out
